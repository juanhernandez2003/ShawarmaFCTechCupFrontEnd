## Contexto del Proyecto
### TechCup
TechCup es un torneo de fútbol interno de la escuela colombiana de ingeniería para las carreras de ingeniería de sistemas, IA, ciberseguridad y estadística por lo que el torneo necesita una aplicación web que organice el torneo, este repositorio se centra principalmente en la parte de FrontEnd de esta aplicación.
